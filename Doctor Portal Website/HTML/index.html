About Us

Welcome to the AsthmAlarm Web Portal! AsthmAlarm is a wearable sensor that addresses nocturnal asthma by using surface electromyography (sEMG) electrodes to monitor and alert patients if they are in danger of an asthma attack. On this website, doctors can register for an account and gain access to a myriad of tools that will make it easier to track their patients. They will be able to register patients and receive daily graphed data of each individual patient after each night.
